# NoveaCockpit

NoveaCockpit is a local-first desktop cockpit for agent work, model routing, interaction review, cost awareness, local knowledge, and audited tool use.

This public repository contains sanitized product material, a fake-data sample GUI, and space for public screenshots. The application source, API, desktop runtime, tests, migrations, local transcripts, reports, logs, model outputs, keys, and operational configuration are private.

## Contents

- `public/`: sanitized product and architecture notes.
- `docs/publication-policy.md`: public/private repository boundary.
- `sample-gui/`: static fake-data GUI preview.
- `sample-gui/images/`: public screenshots approved for sharing.

## Sample GUI

Open `sample-gui/index.html` locally to inspect the static preview. It does not call an API and contains fake data only.

## Repository Boundary

The real app runs privately from a local desktop shell. This repository is only a public product/demo mirror.
