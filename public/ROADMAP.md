# NoveaCockpit Public Roadmap

Last updated: 2026-05-08.

## Current Direction

NoveaCockpit is moving toward a private local desktop app that can coordinate agent runs, inspect real user interactions, connect to local knowledge, and use approved local AI models through Ollama.

## Milestone Themes

- Desktop runtime reliability
- Agent Workbench with approval-gated tool use
- Interaction mirroring from local assistant sessions
- Obsidian knowledge graph onboarding
- Local Ollama model inventory, token tracking, and analysis
- Safe local data tools after audit and approval patterns are proven
- Public documentation mirror separated from private source
- Public fake-data sample GUI and guide for product education

## Bridge Coordination

The secondary Ollama bridge remains a testbed for local model execution, safe wrappers, tool-call audit, artifact visibility, model inventory history, and customer-PC packaging defaults.

Patterns move into NoveaCockpit only after review, parity tests, and explicit acceptance.
